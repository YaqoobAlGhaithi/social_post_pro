# Social Post Pro

![Social Post Pro Banner](https://via.placeholder.com/1200x400.png?text=Social+Post+Pro)

## 🌟 نظرة عامة على المشروع

**Social Post Pro** هو تطبيق ويب متقدم ومبتكر، مصمم لمساعدة صناع المحتوى والمسوقين على تحسين منشوراتهم وتصاميمهم وتدويناتهم لتناسب مختلف الشبكات الاجتماعية. يعتمد التطبيق على قوة الذكاء الاصطناعي التوليدي لتقديم حلول شاملة لتحسين النصوص، ترجمتها، وتوليد صور فنية معبرة تتوافق مع محتوى المنشور. تم تطوير هذا المشروع باستخدام أحدث تقنيات الويب، وتمت إعادة هيكلته لضمان قابلية التوسع والصيانة العالية.

## ✨ الميزات الرئيسية

-   **تحسين المنشورات بالذكاء الاصطناعي:** يقوم بتحليل النص المدخل واقتراح تحسينات لغوية وتسويقية بناءً على المنصة المستهدفة (إنستغرام، لينكد إن، تويتر، فيسبوك) ونبرة الصوت المطلوبة (احترافي، مرح، ملهم، رسمي، تسويقي).
-   **توليد الصور الفنية:** القدرة على توليد صور فريدة وجذابة تتناسب مع المحتوى المحسن للمنشور، مما يضيف قيمة بصرية عالية.
-   **دعم متعدد اللغات:** يدعم تحسين وترجمة النصوص إلى العربية والإنجليزية والفرنسية، مع الحفاظ على دقة المعنى والنبرة التسويقية.
-   **نظام دعم متكامل:** يوفر مركز دعم للمستخدمين يتضمن بطاقة اتصال بالمطور، قسم للأسئلة الشائعة (FAQ)، ومساعد دردشة ذكي للإجابة على الاستفسارات.
-   **واجهة مستخدم بديهية:** تصميم عصري وجذاب يعتمد على مبادئ Brutalism و Neumorphism، مما يوفر تجربة مستخدم فريدة وسهلة الاستخدام.

## 🚀 التقنيات المستخدمة

تم بناء هذا المشروع باستخدام مجموعة من التقنيات الحديثة لضمان الأداء العالي وقابلية التوسع:

| الفئة        | التقنيات المستخدمة                                                                                                  |
| :----------- | :---------------------------------------------------------------------------------------------------------- |
| **الواجهة الأمامية** | React.js, TypeScript, Vite, TailwindCSS, Framer Motion (للرسوم المتحركة)                                        |
| **الخلفية**   | Node.js (لواجهات برمجة التطبيقات الخاصة بالذكاء الاصطناعي - API Endpoints)                                     |
| **الذكاء الاصطناعي** | نماذج الذكاء الاصطناعي التوليدي (مثل Google Gemini API) لتحسين النصوص وتوليد الصور                                |
| **تطبيقات الجوال** | Capacitor.js (لتحويل تطبيق الويب إلى تطبيق أندرويد أصلي), Android Studio (لبناء APK)                           |
| **إدارة الحالة** | React Hooks (useState, useEffect) مع Custom Hooks مخصصة لفصل المنطق                                           |

## 🏗️ هيكلة المشروع

تمت إعادة هيكلة المشروع لاتباع أفضل الممارسات في تطوير React، مما يضمن سهولة القراءة، الصيانة، وقابلية التوسع. الهيكل التنظيمي للمجلد `src/` هو كالتالي:

```
src/
├── data/             # يحتوي على البيانات الثابتة (مثل faqData.ts)
├── hooks/            # يحتوي على الخطافات المخصصة (Custom Hooks) لفصل المنطق البرمجي (مثل useImprover.ts, useSupportChat.ts)
├── components/       # يحتوي على مكونات واجهة المستخدم، مقسمة حسب الوظيفة
│   ├── layout/       # مكونات التخطيط العام (مثل Header.tsx, Footer.tsx)
│   ├── improver/     # مكونات خاصة بوحدة تحسين المنشورات (مثل ImproverModule.tsx, ImproverForm.tsx, ImproverResult.tsx)
│   └── support/      # مكونات خاصة بوحدة الدعم (مثل SupportModule.tsx, SupportSidebar.tsx, ContactHub.tsx, FaqSection.tsx, ChatBot.tsx)
├── App.tsx           # المكون الرئيسي للتطبيق، يجمع المكونات الفرعية ويدير التوجيه الأساسي
├── index.css         # ملفات الأنماط العامة
└── main.tsx          # نقطة دخول التطبيق (React DOM rendering)
```

## 💻 كيفية التشغيل (تطبيق الويب)

للتشغيل المحلي لتطبيق الويب، اتبع الخطوات التالية:

1.  **استنساخ المستودع:**
    ```bash
    git clone https://github.com/YaqoobAlGhaithi/Social-Post.git
    cd Social-Post
    ```

2.  **تثبيت التبعيات:**
    ```bash
    npm install
    ```

3.  **تشغيل خادم التطوير:**
    ```bash
    npm run dev
    ```
    سيتم تشغيل التطبيق على `http://localhost:5173` (أو منفذ آخر متاح).

4.  **بناء المشروع للإنتاج:**
    ```bash
    npm run build
    ```
    سيتم إنشاء ملفات الإنتاج في مجلد `dist/`.

## 📱 كيفية التشغيل (تطبيق أندرويد APK)

لتحويل التطبيق إلى ملف APK وتثبيته على جهاز أندرويد، اتبع الخطوات التالية:

### المتطلبات المسبقة:
-   **Node.js:** مثبت على جهازك.
-   **Java Development Kit (JDK):** يفضل Java 17 أو أحدث (تم استخدام Java 21 في عملية البناء).
-   **Android SDK:** يمكن تثبيته عبر Android Studio أو Android Command Line Tools.

### خطوات البناء:

1.  **تثبيت Capacitor:** تأكد من تثبيت Capacitor في المشروع (تم ذلك بالفعل في المستودع الحالي):
    ```bash
    npm install @capacitor/core @capacitor/cli @capacitor/android
    ```

2.  **تهيئة Capacitor:**
    ```bash
    npx cap init "Social Post Pro" "com.yaqoob.socialpostpro" --web-dir dist
    ```
    (هذه الخطوة تم تنفيذها بالفعل، ولكنها مذكورة هنا للتوثيق).

3.  **إضافة منصة أندرويد:**
    ```bash
    npx cap add android
    ```

4.  **بناء مشروع الويب:**
    ```bash
    npm run build
    ```
    هذه الخطوة ضرورية لتوليد ملفات الويب الثابتة التي سيتم تضمينها في تطبيق الأندرويد.

5.  **مزامنة Capacitor مع مشروع الأندرويد:**
    ```bash
    npx cap sync android
    ```

6.  **بناء ملف APK:**
    -   انتقل إلى مجلد مشروع الأندرويد:
        ```bash
        cd android
        ```
    -   قم بتعيين متغيرات البيئة لـ Java و Android SDK (إذا لم تكن مضبوطة عالمياً):
        ```bash
        export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64 # تأكد من المسار الصحيح لإصدار Java لديك
        export ANDROID_HOME=$HOME/android-sdk # أو المسار الذي قمت بتثبيت SDK فيه
        export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools
        ```
    -   قم ببناء ملف APK (نسخة Debug):
        ```bash
        ./gradlew assembleDebug
        ```
    -   ستجد ملف APK الناتج في المسار التالي:
        `android/app/build/outputs/apk/debug/app-debug.apk`

### تثبيت التطبيق على جهاز أندرويد:

1.  **نقل ملف APK:** انقل ملف `app-debug.apk` إلى جهاز الأندرويد الخاص بك.
2.  **تفعيل مصادر غير معروفة:** تأكد من تفعيل خيار "تثبيت التطبيقات من مصادر غير معروفة" (Install unknown apps) في إعدادات الأمان بجهازك.
3.  **التثبيت:** افتح ملف APK من مدير الملفات على جهازك وقم بتثبيته. قد تظهر رسالة تحذيرية، اختر "التثبيت على أي حال".
